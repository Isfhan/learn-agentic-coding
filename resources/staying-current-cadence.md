# Staying Current Cadence (Step 18 Operationalized)

## Weekly cadence

- **Monday (20 min):** read release notes for Cursor, Claude Code, and MCP ecosystem updates.
- **Wednesday (30 min):** run one micro-experiment on a new pattern/tool.
- **Friday (15 min):** log what worked, failed, and what to adopt.

## Monthly cadence

- Re-score your stack using: Context, Model, Prompt, Tools.
- Archive outdated rules/skills and refresh top 3.
- Publish one artifact update (new eval, hook, skill, or MCP connector).

## Subscriptions

- Cursor docs and changelog
- Anthropic + OpenAI developer updates
- MCP ecosystem repos/newsletters
- AI engineering communities listed in `resources/communities.md`

## Definition of done

- At least 4 logged weekly reviews completed.
- At least 2 shipped improvements to your agent stack per month.
