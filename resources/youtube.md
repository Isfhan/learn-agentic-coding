## 🧭 How to find current content

The YT graph changes fast. Pro tips:

1. Use **sort: recent** (add `&sp=EgIIBQ%253D%253D` to any search URL).
2. Follow 5 creators; let the algorithm bring you adjacent ones.
3. Prefer videos **<3 months old** for tool-specific tutorials — docs change.
