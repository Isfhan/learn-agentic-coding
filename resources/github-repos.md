## 📚 Learn by reading source

Pick one, read its main agent loop. You'll learn more in an hour than a week of tutorials.

1. **smolagents** (`src/smolagents/agents.py`) — tiny, readable.
2. **aider** (`aider/coders/base_coder.py`) — production quality.
3. **modelcontextprotocol/servers/src/github/index.ts** — clean MCP server implementation.
4. **OpenAI Agents SDK** (`src/agents/agent.py`) — official patterns.
